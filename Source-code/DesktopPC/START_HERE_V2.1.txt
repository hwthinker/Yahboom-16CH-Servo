🎯 SAFETY GUIDELINES

DO's ✅:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ Use default delays (especially S2 & S3 = 100ms)
✅ Start with slower delays when testing (150-200ms)
✅ Monitor servo temperature during operation
✅ Stop immediately if you hear buzzing
✅ Let servos cool down if they get warm

DON'Ts ❌:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❌ NEVER set S2/S3 below 50ms (HIGH RISK!)
❌ Don't ignore buzzing sounds (servo struggling!)
❌ Don't operate if servo feels hot
❌ Don't rush when learning (use 150ms+ first)
❌ Don't test at full speed immediately

═════════════════════════════════════════════════════════════════

⚠️ WARNING SIGNS - STOP IMMEDIATELY!

🚨 Servo buzzing           → Delay TOO LOW! Increase now!
🚨 Servo getting hot       → Delay TOO LOW! Increase now!
🚨 Jerky/stuttering move   → Check delay & power supply
🚨 Strange noises          → STOP! Check mechanical issues

═════════════════════════════════════════════════════════════════

🎓 WHY S2 & S3 NEED 100ms?

Beban yang ditanggung:

        ●  ← Gripper (50g)
       /|\
      / | \  ← Wrist (50g)
       |
     ●──●  ← S3 (Elbow) carries 150g+ above it
     |
     |  ← Upper Arm (100g)
     |
     ●  ← S2 (Shoulder) carries 300g+ TOTAL!
     |
   ╔═╗  ← Base

Torque calculation:
S2 Torque = 6-9 kg.cm (near servo limit!)
Fast movement = Torque spike = OVERLOAD!
100ms delay = Safe within spec ✅

🔧 DELAY ADJUSTMENT GUIDE

When to use different delays:

┌─────────┬────────────────────────────────────────────┐
│ Delay   │ When to Use                                │
├─────────┼────────────────────────────────────────────┤
│ 30ms    │ Light joints only (S4, S5, S6)            │
│ 50ms    │ Minimum for S2/S3 (still risky!)          │
│ 100ms   │ DEFAULT for S2/S3 ← RECOMMENDED ✅        │
│ 150ms   │ Extra safe, good for testing              │
│ 200ms+  │ Very safe, learning, heavy loads          │
└─────────┴────────────────────────────────────────────┘

Example scenario:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Testing new pattern:      S2=150ms, S3=150ms (slow & safe)
Daily operation:          S2=100ms, S3=100ms (balanced)
Adding weight to gripper: S2=150ms, S3=150ms (compensate)
Light operations only:    S2=100ms, S3=100ms (don't go lower!)

💡 PRO TIPS

1. Default is your friend
   → Don't change unless you understand why

2. Test before deploy
   → New pattern? Test with 150ms first

3. Add weight = Add delay
   → Heavier gripper/load? +20-50ms on S2/S3

4. Temperature is indicator
   → Warm servo = OK, Hot servo = Problem!

5. Sound matters
   → Buzzing = Warning sign, stop immediately!

6. Better safe than sorry
   → Replacing servo costs 100rb+, delay is FREE!
